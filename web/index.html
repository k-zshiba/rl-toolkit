Explore rl-coach analysis output locally in the browser with a static viewer designed for GitHub Pages.

How it works

  1. Run rl-coach locally to produce analysis/*.json or analysis/summary.json.
  2. Upload one of those JSON files below.
  3. Review the generated report or batch summary without sending data to a server.

Replay harvesting is intentionally omitted here because GitHub Pages cannot safely perform BallChasing API access.
